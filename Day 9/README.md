# Day 9: Euler's Number and Implicit Differentiation

**🚀 Day 9 of the 111 Days of Learning challenge is complete!**

Today I learned why Euler's number `e` is so special, and I explored how to differentiate curves that are defined implicitly:

- ⚡ **Why `e` is special:** The function `e^t` is its own derivative.
- 📈 **Exponential Growth:** How different bases grow at different rates, with `e` standing out naturally.
- 🧮 **Implicit Differentiation:** How to use tiny nudges to find slopes for curves that are not written as `y = f(x)`.
- 🧠 **Hidden Relationships:** How derivatives can reveal structure even when the equation is not solved explicitly.

This made calculus feel more elegant and surprisingly intuitive. 💡
