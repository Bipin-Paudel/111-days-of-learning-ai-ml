# Day 8: Derivative Formulas Through Geometry

**🚀 Day 8 of the 111 Days of Learning challenge is complete!**

Today I explored calculus from a more geometric angle and learned how derivative formulas can be understood visually:

- 📈 **Geometric Derivatives:** How slopes and curves help explain derivative behavior.
- 🌙 **Sine Derivative Intuition:** How `sin(θ)` connects to geometry on the unit circle.
- 〰️ **The `1/x` Example:** How small changes in a function lead to derivative insight.
- 🧠 **Visual Understanding:** How geometry makes derivative rules feel much more intuitive.

This made derivatives feel less abstract and more connected to shapes and motion. 💡
