# Day 3: Inverse Matrices and Dot Products

**🚀 Day 3 of the 111 Days of Learning challenge is complete!**

Today I stayed on the linear algebra path and focused on a few big ideas:

- 🔁 **Inverse Matrices:** How inverses help us "undo" linear transformations and solve systems of equations.
- 📦 **Column Space and Null Space:** How to think about what a matrix can reach, and what gets mapped to zero.
- 🧭 **Dot Products and Duality:** How dot products connect geometry, projection, and measuring alignment between vectors.

These chapters made the bigger structure of linear algebra feel a lot more connected. 💡
