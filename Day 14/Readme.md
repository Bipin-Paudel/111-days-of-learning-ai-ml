# Day 14: Epidemics, Exponential Growth, and Logistic Curves

**🚀 Day 14 of the 111 Days of Learning challenge is complete!**

Today I explored the math behind epidemics and exponential growth:

- 📈 **Exponential Growth:** How continuous multiplication can model rapid spread.
- 🌿 **Logistic Curves:** How growth eventually slows and levels off.
- 🎯 **Inflection Point:** Why the turning point in a logistic curve matters so much.
- 🔧 **Growth Factor:** How lowering the growth factor changes the shape of the curve.

This made growth models feel much more realistic and intuitive. 💡
