# Day 5: Change of Basis, Eigenvectors, and Eigenvalues

**🚀 Day 5 of the 111 Days of Learning challenge is complete!**

Today I explored how matrices can be translated into different coordinate systems and discovered the power of eigenvectors and eigenvalues:

- 🔄 **Change of Basis:** How to move between different coordinate systems and express the same vector in a new language.
- 🧭 **Basis Transformations:** How a transformation matrix changes when we switch the basis we use to describe it.
- ⚡ **Eigenvectors and Eigenvalues:** How certain directions stay special under a transformation, and how scaling factors reveal what the matrix is really doing.
- 🧠 **Eigenbasis:** How diagonalizing a transformation makes repeated computation much easier and cleaner.

This made linear algebra feel even more elegant and connected. 💡
