# Day 11: Fixed Points and Taylor Series

**🚀 Day 11 of the 111 Days of Learning challenge is complete!**

Today I explored two powerful ideas from calculus:

- 🔁 **Fixed Points:** How repeated inputs can settle toward a stable value, like with `f(x) = 1 + 1/x`.
- 📉 **Derivative Magnitude:** Why a small derivative can hint that iteration will converge.
- 🧠 **Taylor Series:** How a function can be approximated using its value and all of its derivatives at one point.
- ⚡ **Higher-Order Derivatives:** How the first, second, third, and higher derivatives all contribute to the approximation.

This made calculus feel even more like a tool for understanding behavior, not just computing numbers. 💡
