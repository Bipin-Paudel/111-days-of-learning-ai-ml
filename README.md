# 111 Days of Learning - AI/ML Journey

A structured 111-day learning journey focused on Artificial Intelligence and Machine Learning, built for the Code for Change challenge.

This repository is my public learning log for:
- daily AI/ML practice
- hands-on Python and notebook work
- notes from videos, articles, and courses
- small projects and experiments
- progress tracking throughout the challenge

## Why this repo exists

The goal is simple: learn consistently, build publicly, and improve every day.

This repo follows the challenge spirit:
- learn something new every day
- share progress publicly
- stay consistent
- keep notes organized
- build real skills over time

## What you will find here

- daily learning notes
- code snippets and practice notebooks
- resource links
- mini projects
- summaries of what I learned each day
- progress updates for the 111-day challenge

## Learning Roadmap

This journey follows a structured AI/ML roadmap:

1. Mathematics and Statistics
2. Python Mastery

5. Deep Learning
6. NLP and Computer Vision
7. MLOps and Production Tools
8. Data Scientist Role Skills
